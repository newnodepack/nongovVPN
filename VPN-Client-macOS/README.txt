VPN Client for macOS
========================================

Setup:
  1. Edit .env with your server info
  2. Double-click 'VPN-Client.command'
  3. Enter admin password when prompted

Alternative:
  cd /path/to/VPN-Client-macOS
  sudo ./VPN-Client

If blocked by Gatekeeper:
  xattr -cr /path/to/VPN-Client-macOS/*
